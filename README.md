## github简易使用说明

#### 首次从远程仓库克隆至本地文件夹：
>`git clone https://github.com/andy1li/taobao-page-clone.git`

#### 将远程仓库的最新改动同步至本地（要经常使用哦）：
>`git pull`

#### 在自己修改本地代码后，推送至远程仓库：
>`git add .` *（先将改动的文件加入缓冲区）*

>`git commit -m "这里替换成改动描述"` *（再实际提交改动，并描述相关改动）*

>`git push` *（最后推送至远程仓库，需要根据提示输入github用户名和密码）*
